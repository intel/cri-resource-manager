Contributing¶

Please use the GitHub* infrastructure for contributing to CRI Resource Manager. Use pull requests to contribute code, bug fixes, or if you want to discuss your ideas in terms of code. Open issues to report bugs, request new features, or if you want to discuss any other topics related to CRI Resource Manager or orchestration resource management in general.
