Configuration Reference

Configuration file

WORK IN PROGRESS

policy

Active specifies the active policy.

policy:
  Active: static

AvailableResources specifies the available hardware resources.

ReservedResources specifies the hardware resources reserved for system and kube tasks.

Currently, only CPU resources are supported. CPUs may be specified as a cpuset or as a numerical value, similar to Kubernetes resource quantities. Not all policies use these configuration settings. See the policy-specific documentation for details.

policy:
  AvailableResources:
    cpu: cpuset:0-63
  ReservedResources:
    cpu: cpuset:0-3
    # Alternative ways to specify CPUs:
    #cpu: 4
    #cpu: 4000m

policy.static

RelaxedIsolation controls whether isolated CPUs are preferred for Guarenteed Pods.

policy:
  static:
    RelaxedIsolation: true
